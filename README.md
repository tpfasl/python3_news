## ✨ 1. 프로젝트 목표
키워드를 입력하면 관련된 뉴스를 자동으로 수집하고 요약하여, **핵심 문장만 출력**하는 **맞춤형 뉴스 요약 시스템** 개발

> 사용자가 원하는 정보에 빠르게 접근하고, 뉴스 소비 시간을 혁신적으로 단축하는 것을 목표로 합니다.

## 🚀 2. 주요 기능
제출하신 목표 결과물 초안을 바탕으로 기능을 정리했습니다.

### 2.1. 키워드 뉴스 수집 (News Collection)
* **키워드 기반 구글 뉴스 검색:** 사용자가 입력한 키워드를 기반으로 구글 뉴스에서 관련 기사 제목과 링크를 자동으로 수집합니다.
* **최신 뉴스 가져오기 :** 최대 15개의 최신 뉴스를 가져와 콘솔에 표시합니다.
* **제목 정제:** 각 기사의 제목을 정제하여 불필요한 문장/표현을 제거합니다.

### 2.2. 자동 요약 및 결과 저장 (Summarization & Save)
* **핵심 문장 기반 요약:** 뉴스 제목 및 본문을 **핵심 문장 중심**으로 간단히 요약합니다.
* **주요 키워드 추출:** 단어 빈도 분석(TF-IDF 등)을 통해 기사 내용의 **주요 키워드 상위 7개**를 추출하여 출력합니다.
* **자동 파일 저장:** 요약 결과와 통계 정보를 **텍스트 파일(.txt)** 로 자동 저장합니다.

## 💡 3. 기대 효과 (Expected Effects)
| 구분 | 내용 |
| :--- | :--- |
| **정보 접근성 향상** | 사용자가 원하는 키워드 기반의 관련 뉴스를 **한눈에** 볼 수 있어 필요한 정보에 대한 접근성이 높아집니다. |
| **뉴스 소비 시간 단축** | 긴 기사 대신 핵심 문장만 요약하여 **짧은 시간** 안에 주요 내용을 파악할 수 있습니다. |
| **개인 맞춤 정보 제공** | 사용자의 관심 키워드를 기반으로 **개인화된** 뉴스 요약을 제공합니다. |
| **뉴스 흥미도 증가** | 요약된 뉴스는 가독성이 높아 평소 뉴스에 관심이 적던 사용자도 자연스럽게 흥미를 가질 수 있습니다. |
